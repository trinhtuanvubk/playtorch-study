# playtorch-study
Study playtorch for mobile


### Installing

- To install nodejs and npm
```bash
sudo apt update
sudo apt install nodejs npm
nodejs --version
```

- To install `yarn`:
```bash
sudo npm install -global yarn
yarn --version
```

- To create an example app: 
    - with `npx`
        ```
        npx create-expo-app --template
        ```
    - with  `yarn`
        ```
        yarn create expo
        ```
